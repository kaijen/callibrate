# Detail-Ansicht

Ab Version 0.6.0 öffnet das Antippen einer aufgelösten Vorhersagenkarte eine Detail-Ansicht.

## Inhalte

Die Detail-Ansicht zeigt:

- Fragentext
- Kategorie (`epistemic` oder `aleatory`)
- Tags
- Schätzung (Wahrscheinlichkeit, Ja/Nein-Wahl mit Konfidenz, oder Intervall)
- Auflösung: Ergebnis (eingetreten / nicht eingetreten), Notizen, numerischer Messwert (bei Intervall-Typ)

## Aufruf

Auf eine **aufgelöste** Vorhersagenkarte tippen. Offene und ausstehende Karten führen weiterhin zu Schätz- und Auflösungsscreen.
